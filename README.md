# 离线推理 (Just a Snippet)

使用 vLLM 批量发起离线推理，手动实现数据并行，提高推理效率。

## 快速开始

可参考 `tasks` 目录下的 `complexbench` 示例。

### 使用步骤

1. **编辑数据处理逻辑**：根据任务需求编辑数据输入和输出解析函数
2. **配置推理参数**：在 `run_parallel.sh` 中设置默认推理参数
3. **批量执行**：使用 `batch-run_parallel.sh` 覆盖部分默认参数，完成批量调用

## 特性

- 基于 vLLM 的高效离线推理
- 手动数据并行，支持多 GPU 推理
- 灵活的参数配置和批量执行
